写作工作流推荐

一套高效的公众号写作流程：

为什么选择 Markdown + MarkCopy？

  1. 专注内容：Markdown 语法简洁，写作时不被排版分心
  2. 一次编写，多端发布：同一篇文章可以发公众号、知乎、掘金，格式自动适配
  3. 版本管理：纯文本格式，方便用 Git 管理文章版本
  4. 排版一致：告别公众号编辑器的格式错乱问题
